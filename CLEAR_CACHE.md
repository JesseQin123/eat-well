# 完全清除缓存指南

## Chrome/Edge 浏览器

### 方法 1：开发者工具（推荐）
1. 按 `F12` 打开开发者工具
2. **右键点击** 浏览器刷新按钮
3. 选择 "清空缓存并硬性重新加载"

### 方法 2：Application 标签
1. 按 `F12` 打开开发者工具
2. 转到 **Application** 标签
3. 左侧菜单：
   - **Service Workers** → 点击 "Unregister"
   - **Cache Storage** → 右键删除所有缓存
   - **Local Storage** → 可选清除
4. 关闭开发者工具
5. 硬刷新：`Cmd+Shift+R` (Mac) 或 `Ctrl+Shift+R` (Windows)

### 方法 3：隐身模式（最简单）
1. 按 `Cmd+Shift+N` (Mac) 或 `Ctrl+Shift+N` (Windows)
2. 在隐身窗口访问 http://localhost:5177/
3. 这样会跳过所有缓存

## Safari 浏览器
1. 按 `Cmd+Option+E` 清空缓存
2. 或者菜单栏 → 开发 → 清空缓存
3. 硬刷新：`Cmd+Shift+R`

## Firefox 浏览器
1. 按 `Ctrl+Shift+Delete` 打开清除历史记录
2. 选择 "缓存"
3. 点击 "立即清除"
4. 硬刷新：`Ctrl+Shift+R`
